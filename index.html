Matrices are fun

This is the dissertation project of UP938279. It is supervised by Dr John Geddes of the University of Portsmouth.

This website is designed to help you learn matrices. It is not a true standalone tool - it is to assist with other forms of teaching. It is recommended that you start with what a matrix is, before moving on to adding or subtracting matrices. The contents list below suggests in what order you should visit the pages of this website.

If you are feeling confident in your knowledge, perhaps you would like to test it - use the quiz feature.
